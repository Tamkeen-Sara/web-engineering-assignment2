F. FIGMA IMPLEMENTATION CHECKLIST

Step-by-Step Implementation Guide

1. File Setup

  • ☐ Create new Figma file: "TechConnect — MVP Wireframes"
  • ☐ Create 3 pages: Design Tokens, Wireframes, Prototype
  • ☐ Add color swatches on Design Tokens page with hex codes
  • ☐ Add typography specs and button styles

2. Layout Grids & Frames

  • ☐ Create Desktop frame: 1440 × 1024px
  • ☐ Add layout grid: 12 columns, 96px margins, 24px gutter
  • ☐ Duplicate and name: Home, Browse, PostProject, Dashboard

3. Build Components

  • ☐ Header component (72px height, sticky)
  • ☐ Company Card component (320×160px)
  • ☐ Company List Card (horizontal, 140px height)
  • ☐ Template Card (240×120px)
  • ☐ Button variants (Primary/Secondary, Default/Hover/Disabled)
  • ☐ Input variants (Default/Error/Focused)
  • ☐ Verified Badge (24×24px circle, teal #008C7E)
  • ☐ Checkbox, Radio, Toggle components

4. Build All Pages

  • ☐ Home: Hero, categories, featured companies, steps, templates, footer
  • ☐ Browse: Sidebar filters (300px), company list cards, compare bar
  • ☐ Post Project: Two-column form (8+4 cols), template selector, help sidebar
  • ☐ Dashboard: 3-col layout (240px nav + main + 300px sidebar)

5. Prototype Linking

  • ☐ Header logo → Home
  • ☐ "Explore Companies" button → Browse
  • ☐ Category chips → Browse (with filter active)
  • ☐ "Post Project" button → Post Project page
  • ☐ "Use Template" → Post Project (template selected)
  • ☐ "Publish Project" → Confirmation modal
  • ☐ "Invite to Bid" → Modal overlay
  • ☐ Project cards → Dashboard details
  • ☐ Set transitions: Smart Animate or Instant

6. Extra UI Details

  • ☐ Verified badge tooltip: "Verified by TechConnect — Company registration & reference checks completed"
  • ☐ Language toggle behavior (EN/اردو)
  • ☐ Project templates with prefilled text
  • ☐ Payment icons: JazzCash, EasyPaisa placeholders
  • ☐ Company verification CTA banner

7. Testing & Export

  • ☐ Click "Present" to test all flows
  • ☐ Verify all interactions work
  • ☐ Check spacing and alignment
  • ☐ Share → "Anyone with link can view"
  • ☐ Export screenshots: Home, Browse, Post Project, Dashboard
  • ☐ Optional: Create mobile versions (375×812px)

📝 Important Notes for Wireframes

  • Grayscale Only: Wireframes use blacks, grays, whites — no brand colors
  • Dashed Borders: Use dashed borders for placeholders (logos, images)
  • Simple Shapes: Rectangles, circles, basic shapes — no detailed styling
  • Placeholder Text: Use generic labels like "Logo", "Image", "Text Block"
  • Focus on Structure: Show WHERE things go, not HOW they look
  • Dimensions Matter: Include exact pixel dimensions as annotations
  • Grid System: Always work within the 12-column grid system
  • Component Specs: Document all component sizes and spacing

🎨 Prototype Differences (Task 4)

  • Full Brand Colors: Navy #0A2540, Teal #008C7E, Orange #FF8A2B
  • Real Typography: Poppins 600 for headings, Inter 400 for body
  • Actual Branding: Real TechConnect logo, not placeholder
  • Styled Components: Fully designed buttons, cards, badges
  • Interactive: Clickable with working navigation flows
  • Real Content: Actual company names, realistic text
  • Visual Effects: Shadows, hover states, transitions
